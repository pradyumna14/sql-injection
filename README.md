# Abstract
SQL injection attacks remain one of the most critical threats to database-driven applications, often bypassing traditional rule-based security mechanisms. This paper presents a deep learning-based approach for detecting SQL injection attacks by learning from large-scale datasets of malicious and benign queries. The proposed system leverages sequence-based neural network models to capture structural and semantic patterns within query inputs, enabling robust classification of both known and previously unseen attack variants. Feature representations are engineered to preserve contextual relationships in query syntax, improving the model’s ability to generalize across diverse attack patterns. The system is implemented using Python/C++ and evaluated on benchmark datasets, demonstrating effective detection performance and resilience against zero-day SQL injection attempts. The results highlight the potential of deep neural networks in enhancing automated and adaptive database security systems.
